# Passo 3 — Interface de chat (front-end)

## Objetivo
Construir a experiência principal do produto com foco em simplicidade.

## Funcionalidades mínimas
- Campo de entrada fixo no rodapé
- Lista de mensagens em ordem cronológica
- Estados visuais: enviando/processando/concluído/erro
- Histórico por sessão
- Renderização de markdown e blocos de código

## Requisitos de UX
- Layout responsivo
- Rolagem automática para última mensagem
- Mensagens de erro amigáveis
- Indicador de ferramenta em uso (opcional no MVP)

## Entregáveis
- Protótipo de baixa fidelidade
- Componentes base (Chat, MessageList, Input)
- Tela funcional conectada à API de chat

# Passo 6 — Fluxos n8n

## Objetivo
Automatizar integrações e tarefas repetitivas com workflows versionáveis.

## Fluxos recomendados para V1
1. Receber tarefa do backend
2. Enriquecer contexto (opcional)
3. Chamar provedor de IA
4. Pós-processar resposta
5. Registrar logs e retornar resultado

## Boas práticas
- Nomear workflows com padrão consistente
- Versionar export dos fluxos
- Definir nós de erro e retentativa

## Entregáveis
- Workflow principal de chat
- Workflow de fallback
- Guia de operação dos fluxos
